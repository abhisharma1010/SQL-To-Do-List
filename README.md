# SQL-To-Do-List

A simple SQL-based To-Do List application demonstrating hands-on SQL skills including database creation, table design, CRUD operations, and query execution.

## Features

This SQL script includes:

1. Database Creation

Creates a database named todo_app

2. Table Schema

Creates a table tasks with:

id (Primary Key, Auto Increment)

description (Task details)

status (Default: 'Pending')

3. CRUD Operations

Create → Insert sample tasks

Read → Select all tasks

Update → Mark task as Done

Delete → Remove a task
